# Quality Checklist

## Feature Functionality
- [ ] Feature works as described in the spec
- [ ] All input validation works
- [ ] Error states are handled gracefully

## UI Consistency (New Standard)
- [ ] **Header**: Verfiy gradient styling and font size match Analytics/Insights.
- [ ] **Container**: Verify background color, padding, and border-radius match other pages.
- [ ] **Responsive**: Verify layout on mobile and desktop.
- [ ] **Dark Mode**: Verify appearance in dark mode.

## Platform & Performance
- [ ] Works in Electron
- [ ] No console errors
